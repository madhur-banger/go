Go Web Server Project

A beginner-friendly project that demonstrates how to build a web server using Golang.

Project Overview

This project is a simple web server built using the Go programming language. The web server serves static HTML files and handles multiple HTTP routes, demonstrating how a web server interacts with client requests like GET, POST, PUT, and DELETE.

Features

  • Serves static HTML files, such as index.html and form.html
  • Handles HTTP GET requests to serve a "Hello" message
  • Handles HTTP POST requests to submit and display form data
  • Supports HTTP PUT requests to update user information
  • Supports HTTP DELETE requests to remove user information

How It Works

The project consists of a Go web server that listens on port 8080. It serves static files from the ./static directory and processes requests based on the route:

  • /hello: A GET route that responds with "Hello, welcome!"
  • /form: A POST route that accepts form submissions and prints the user's name and address
  • /update: A PUT route that updates user information
  • /delete: A DELETE route that removes a user

Each request is handled by specific handler functions in the main.go file, which processes the request and sends an appropriate response back to the client.

Technologies Used

  • Go Language (Golang): The primary programming language used for building the web server.
  • HTML5: Used for creating static web pages (e.g., index.html and form.html).
  • HTTP: Protocol used to handle GET, POST, PUT, and DELETE requests between the client and server.

Folder Structure

  • go-server/: The main project directory containing all the files.
  • main.go: The main Go file where the server is defined and routes are handled.
  • static/: Directory containing static HTML files.
    • index.html: The main landing page of the website (this file).
    • form.html: The form used to submit data via a POST request.

How to Run the Project

Follow these steps to run the project on your local machine:

  1. Make sure you have Go installed on your system.
  2. Navigate to the project directory:
    3. cd go-server
  3. Run the web server using the command:
    4. go run main.go
  4. Open your browser and go to http://localhost:8080 to access the site.
